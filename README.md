# Excel Projects Portfolio

This repository contains all my **Excel-based data analytics projects**.  
Projects are organized by category (Sales, HR, Finance, Marketing) for easy navigation. Each project folder contains:

- Excel dashboard / analysis file (`.xlsx`)  
- Dashboard screenshot (`.png`)  
- Project-specific `README.md` with detailed insights

## Project Categories

Folder: [Sales-Analytics](https://github.com/damaksharma/excel-projects/tree/main/sales-analytics)

## How to Explore
1. Click on a category folder above.  
2. Open a project folder to see:  
   - `README.md` → Detailed description, analysis steps, insights  
   - Excel file (`.xlsx`) → Dashboard or analysis  
   - Dashboard screenshot (`.png`)
