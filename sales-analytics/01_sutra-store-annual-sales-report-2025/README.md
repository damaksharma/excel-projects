# Sutra Store Sales Analysis Dashboard (2025)

## Project Overview

This project analyzes the annual sales performance of **Sutra Store** using Microsoft Excel.
The dataset contains more than **15,000 rows and 20 columns** of retail sales data.

The goal of this project was to clean the data, analyze key business metrics, and build an **interactive dashboard** that helps understand customer behavior, sales channels, and regional performance.

---

## Dataset

* Retail sales dataset
* 15,000+ rows
* 20 columns
* Includes information about orders, customers, gender, age group, order channel, and sales performance.

---

## Data Preparation

The dataset required preprocessing before analysis.

Steps performed:

* Data cleaning
* Handling missing values
* Standardizing categorical values
* Formatting date and numeric fields
* Preparing the dataset for Pivot Table analysis

---

## Analysis Performed

Using **Pivot Tables**, the following business questions were analyzed:

* Sales vs Orders performance
* Purchase distribution between **Men vs Women**
* Order status distribution
* Customer segmentation by **Age Group and Gender**
* Performance comparison across **Order Channels**

---

## Dashboard Features

An **interactive Excel dashboard** was created using:

* Pivot Charts
* Slicers
* Dynamic filtering
* Clean visual layout for easy insights

The dashboard allows users to quickly explore sales data and filter results by different dimensions.

---

## Key Business Insights

* **Women contributed approximately 69% of total orders**
* **Amazon was the highest performing sales channel**
* **Maharashtra generated the highest sales revenue**
* Customer segmentation showed strong purchasing activity among specific age groups.

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts

* Data Cleaning
